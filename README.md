# bakhtishod — Responsive Portfolio

This is a simple responsive portfolio website built with Bootstrap for the LimeLight TechLead internship assignment.

Pages:
- `index.html` — single-page site with sections: Services, Portfolio, About, Academics, Favorites, Contact.

Deploy to GitHub Pages:
1. Push this repository to GitHub.
2. In repository settings > Pages, choose branch `main` (or push to `gh-pages`) and root or `/docs` depending on setup.
3. Optionally add a `CNAME` file with your custom domain and push.
